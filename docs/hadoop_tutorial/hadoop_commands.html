Frequently Used Hadoop Commands

  1. Format the HDFS.

    2. /hadoop$ bin/hdfs namenode -format
  2. Creating the HDFS directories required to execute MapReduce jobs.

    3. /hadoop$ bin/hdfs dfs -mkdir /user

    /hadoop$ bin/hdfs dfs -mkdir /user/<username>

    #Replace <username> accordingly
  3. Copying files to HDFS.

    4. #This command will put files to default HDFS directory (/user/<username>)

    hadoop$ bin/hdfs dfs -put path-to-file-in-local-filesystem


    #This command will copy files to the destination directory of HDFS.
    #The below command will copy a file to root of the HDFS

    hadoop$ bin/hdfs dfs -copyFromLocal path-to-file-in-local-filesystem /
  4. Removing a file or directory

    5. #This command will delete a directory on HDFS

    hadoop$ bin/hdfs dfs -rmdir /user/<username>/dir


    #This command will delete a file on HDFS. (In this example, foo.txt file will be deleted)

    hadoop$ bin/hdfs dfs -rm /user/foo.txt
